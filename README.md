# Twitter
Somos o grupo 7 e levantamos os Requisitos de Software para o app do Twitter.

# Instruções

Requisitos:
```
Python, pip
```
Como Instalar:
```
pip install mkdocs
pip install mkdocs-bootswatch
```
* pode ser necessário permissões de administrador

Como Executar:
```
clone o repositório
cd twitter_software_requirements
mkdocs serve
```
