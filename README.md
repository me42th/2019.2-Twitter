# Instruções

Requisitos:
```
Python, pip
```
Como Instalar:
```
pip install mkdocs
pip install mkdocs-bootswatch
```
* pode ser necessário permissões de administrador

Como Executar:
```
clone o repositório
cd 2019.2-Twitter
mkdocs serve
```
