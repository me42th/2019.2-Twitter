Cenários 1.0

Esta página apresenta a última versão de todos os cenários modelados para este projeto com o app Twitter.

Um cenário é uma estratégia reconhecida para compreender as interações entre ambiente e sistema, bem como elicitar a parte comportamental do software, sua dinâmica e/ou seu fluxo.

Agora os cenários desta página se apresentam em um formato de tabela com cada ocorrência de léxico sendo um link para a sua definição.

Alguns cenários para o app Twitter

Trending Topics

Autor : Erick Giffoni
Versão: 0.2

Titulo: trending topics
Objetivo: Acompanhar quais são os assuntos mais falados no twitter em determinado momento.
Contexto: Navegação no app twitter
Atores: Usuário
Recursos: - Servidores do twitter devem estar funcionando;
- Outros usuários precisam estar falando sobre algum assunto em comum;
- Usuário deve acessar o app twitter;
- Usuários devem usar uma palavra-chave sobre o assunto ou uma hashtag (#) para identificar o tema do assunto.
Exceção: - App não abre;
- Usuário tem o celular roubado;
- Não há assuntos sendo discutidos no momento;
- Servidores do twitter param de funcionar.
Episódios: - Usuário abre o app twitter;
- Usuário clica no ícone de pesquisa;
- Usuário seleciona "trending";
- Usuário escolhe um dos assuntos e acompanha os tweets;
- Usuário pesquisa um assunto;
- Usuário pesquisa uma hashtag.

Login

Autor : Bruno Duarte
Versão: 0.2

Titulo: Login
Objetivo: Autenticar e possibilitar o acesso a plataforma do twitter.
Contexto: Contato inicial com a aplicação.
Atores: Usuário, Sistema
Recursos: - Acesso a Internet;
- Email ou Telefone Móvel;
- Aplicativo do twitter ou Desktop.
Exceção: - Sem acesso a Internet;
- Baixa nos Servidores da Aplicação;
- Aplicativo restringido.
Episódios: - Usuário baixa o aplicativo;
- O usuário cede suas credenciais;
- O sistema autentica as informações do usuário;
- O usuário consegue acesso a plataforma e todas as suas funcionalidades.

Versão: 0.2

Titulo: enviar mensagem
Objetivo: Comunicar-se diretamente com outro usuário do twitter

Criar Tweets

Autor : Lorrany Azevedo
Versão: 0.2

Titulo: criar tweets
Objetivo: Criar publicações que podem ser textos, imagens, vídeos, gifs, emojis, entre outros.
Contexto: Navegação pelo app do twitter
Atores: Usuários
Recursos: - Servidores do twitter;
- Computador, celular ou tablet;
- O usuário precisa estar conectado a internet;
- O usuário precisa estar logado em sua conta do twitter.
Exceção: - Queda dos servidores;
- Problemas na internet;
- Problemas no hardware que está sendo utilizado.
Episódios: - Usuário abre a aplicação do twitter;
- Usuario clica em "criar novo tweet";
- Usuário escreve o que irá publcicar;
- Usuário clica em "tweetar" para publicar o seu tweet.

Explorar

Autor : Fernando Aguilar
Versão: 0.2

Titulo: explorar
Objetivo: Descobrir novos conteúdos com base nas categorias de sugestões feitas pelo twitter.
Contexto: - Navegação no app twitter;
- O usuário pode navegar por novos tweets e explorar assuntos de acordo com as categorias pré-definidas.
Atores: Usuário
Recursos: Acessar sugestões para você, Notícias, Esportes, Divrersão, Entreterimento, trending topics, etc.
Exceção: - O app não abre;
- Não há notícias recomendadas;
- O usuário não possui conexão com a rede e portanto as notícias não são exibida.
Episódios: - O usuário abre o app twitter;
- Ele clica no icone explorar;
- O usuário escolhe um tópico da lista para ler;
- Ele clica no tópico e é redirecionado para a nova página.
