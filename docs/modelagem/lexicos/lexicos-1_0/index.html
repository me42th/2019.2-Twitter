Léxicos 1.0

Esta página apresenta a última versão de todos os léxicos modelados para este projeto com o app Twitter.

Agora os léxicos desta página se apresentam em um formato de tabela com cada ocorrência de léxico sendo um link para a sua definição.

LAL - Léxico Ampliado da Linguagem

Trata-se de uma técnica que compõe os cenários de forma a descrever os símbolos de uma linguagem.
Nesse caso, essa técnica descreve alguns termos relacionados ao software em questão.

Cada símbolo tem um nome, uma noção e um impacto, sendo que :

  • noção : é o que significa o símbolo (denotação);
  • impacto : é o efeito do símbolo na aplicação ou o efeito de algo na aplicação sobre o símbolo (conotação).

Símbolos também possuem uma classificação, que pode ser : estado ; verbo ; objeto ; sujeito.

Alguns léxicos para o app Twitter

Trending

Autor : Erick Giffoni
Versão: 0.2

Nome: trending
Noção: - Aba no aplicativo twitter a qual reune um conjunto de assuntos que estão sendo mais discutidos no momento, seja por meio de hashtags (#) ou por meio de palavras-chave;
- Quem reune essas informações é o algoritmo do app twitter;
- Cada assunto mais falado reune um conjunto de tweets
Classificação: objeto
Impacto(s): - O usuário pode interagir com os assuntos mais falados (trending topics);
- O usuário pode realizar um tweet relacionado a algum trending topic;
Sinônimo(s): assuntos mais falados, tendências.

Tweet

Autor : Erick Giffoni
Versão: 0.2

Nome: tweet
Noção: - Tarefa realizada pelo usuário;
- Acontece quando o usuário quer compartilhar alguma informação no app twitter;
- É como uma caixa. Nela encontram-se informações sobre o usuário que a fez, há quanto tempo foi feita, o conteúdo da caixa (a informação em si) , além da quantidade de curtidas (ou likes), quantidade de comentários e quantidade de compartilhamentos (retweets);
Classificação: objeto
Impacto(s): - O tweet está disponível para ser visualizado;
- O tweet pode ser apagado;
- O tweet pode ser denunciado;
- Outros usuários podem interagir com o tweet.
Sinônimo(s): post, postagem, retweet.

Efetuar login

Autor : Bruno Duarte
Versão: 0.2

Nome: efetuar login
Noção: - Ter acesso à conta registrada;
- Acesso a plataforma.
Classificação: verbo
Impacto(s): - Usuário efetua login na aplicação.
Sinônimo(s): logar, acessar, entrar.

Interações com Postagens

Autor : Bruno Duarte
Versão: 0.2

Nome: interação com postagens
Noção: - Usuário pode favoritar(like) postagens de outros usuários.
- Ocorre quando o usuário deseja interagir com postagens de outros perfis expressando a sua opinião.
- Usuário deseja compartilhar determinada postagem ou comentário para que esteja presente em seu próprio perfil.
Classificação: estado
Impacto(s): - Maior comunicação com usuários da plataforma;
- Liberdade de expressão do usuário na aplicação;
- O conteúdo compartilhado ou comentado pode ser denunciado (tweet, posts, imagens).
Sinônimo(s): share, like, retweetar, comentar, favoritar, compartilhar, likes, posts, postagens, retweet, coments, likar, fav.

Notificações

Autor : Eugênio Sales
Versão: 0.2

Nome: notificações
Noção: Ato ou efeito de levar a alguém o conhecimento de interações de outros usuários relacionadas ao seu perfil como:
- Novos Seguidores;
- likes;
- Menções.
Classificação: objeto
Impacto(s): - Usuário gera interatividade na plataforma;
- Usuário aumenta sua rede de seguidores;
- Perfil do usuário ganha relevância.
Sinônimo(s): menções, atualizações.

Seguidor

Autor : Eugênio Sales
Versão: 0.2

Nome: seguidor
Noção: - Usuário que quer receber conteúdo de outro perfil;
- Usuário que quer se manter engajado com o conteúdo de outro usuário.
Classificação: sujeito
Impacto(s): - Quanto mais seguidores um perfil possui, mais influente ele se torna;
- O conteúdo de um perfil é distribuído para a Home de outros perfis.
Sinônimo(s): rede de seguidores.

Direct Message

Autor : Lorrany Azevedo
Versão: 0.2

Nome: direct message
Noção: - Tarefa realizada pelo usuário;
- Acontece quando o usuário deseja enviar uma mensagem privada para outro;
- É como um chat para os usuário do twitter terem conversar privadas com seus seguidores.
Classificação: objeto
Impacto(s): Mensagens que só serão visíveis para quem estiver na conversa.
Sinônimo(s): mensagem, mensagem direta, dm.

Tweetar

Autor : Lorrany Azevedo
Versão: 0.2

Nome: tweetar
Noção: - Tarefa realizada pelo usuário;
- Acontece quando o usuário deseja publicar um tweet;
- É um botão que pode ser clicado pelo usuário para que seu tweet seja publicado.
Classificação: verbo
Impacto(s): Torna um tweet disponível para que outros usuários possam vê-lo.
Sinônimo(s): publicar, postar.

Perfil

Autor : Fernando Aguilar
Versão: 0.2

Nome: perfil
Noção: Conjunto das informações pessoais que identificam o usuário (nome, idade, profissão, cidade de nascimento etc.) e fica disponível em redes sociais.
Classificação: objeto
Impacto(s): - Representação do objeto;
- Reunião de todas as qualidades pessoais ou profissionais de algúem.
Sinônimo(s): profile, página de usuário.

Para Você

Autor : Fernando Aguilar
Versão: 0.2

Nome: para você
Noção: Sugestões de conteúdo personalizado baseado no conteudo que você curte e nos seus amigos
Classificação: estado
Impacto(s): O sistema recomenda conteúdos populares
Sinônimo(s): sugestões, recommendation.

Twitter

Autor : Aline Laureano
Versão: 0.2

Nome: twitter
Noção: É uma rede social e um servidor para microblogging.
Classificação: objeto
Impacto(s): Além de permitir um usuário realizar cadastro e login, o Twitter permite aos seus usuários enviar e receber atualizações pessoais de outros contatos, por meio da plataforma web do serviço, por SMS e por softwares específicos de gerenciamento.
Sinônimo(s): Twitter, TWITTER.

Hashtag

Autor : Aline Laureano
Versão: 0.2

Nome: hashtag
Noção: São palavras-chave ou termos associados a uma informação, tópico ou discussão que se deseja indexar de forma explícita no twitter. Ela é iniciada pelo símbolo "#".
Classificação: objeto
Impacto(s): As hastags permitem o agrupamento de tweets e informações que obtenham hashtags em comum.
Sinônimo(s): #hashtag, Hashtag.
